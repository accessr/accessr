Alt‑Text for Word: A Fast, Practical Checklist

Published 8 November 2025

  1. State the point. Describe what the image is conveying, not every pixel.
  2. Be concise. Aim for 1–2 sentences; skip “image of…”.
  3. Note text in images. If an image contains essential text, include it.
  4. Mark decorative images. If truly decorative, leave alt empty and mark as decorative.
  5. Avoid redundancy. Don’t repeat captions verbatim unless essential for context.
